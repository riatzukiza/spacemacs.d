This directory contains the personal [[https://www.spacemacs.org/][Spacemacs]] configuration.

* Files

- =init.el= :: Main Spacemacs configuration file (formerly =~/.spacemacs=).
- =.spacemacs.env= :: Environment variables loaded at startup.
- =layers/= :: Custom Spacemacs layers.
- =gptel-tools.el= :: Additional Elisp utilities for =gptel=.

* Layout

Spacemacs loads =init.el= from this directory when =~/.spacemacs.d/init.el= exists.
Custom layers are resolved through the path set in
=dotspacemacs-configuration-layer-path=:

#+begin_example elisp
dotspacemacs-configuration-layer-path '("~/.spacemacs.d/layers/")
#+end_example

* Testing

Run the full ERT suite locally:

#+begin_src sh
make test
#+end_src

Generate a coverage report (requires Emacs with MELPA access):

#+begin_src sh
make coverage
# report written to coverage/lcov.info
#+end_src

CI runs on every push and pull request via GitHub Actions. PRs are blocked
from merging if any test fails. Coverage artifacts are uploaded per PR.

** Adding tests

- Unit tests live in =tests/unit/=. Each file targets one layer's =funcs.el=.
- Integration tests live in =tests/integration/=. Each file targets one layer's
  declared package list and init function contracts.
- Smoke tests live in =tests/smoke/=. These assert startup-level invariants.

Name every =ert-deftest= with the =μ/= prefix and make the name read as a claim:

#+begin_src elisp
(ert-deftest μ/my-func-does-the-right-thing ()
  ...)
#+end_src
