#+TITLE: Custom Spacemacs Layers
#+AUTHOR: err
#+DATE: 2026-06-12

This directory holds custom Spacemacs layers used by the configuration in
~/.spacemacs.d/init.el=.

* Layer index

| Layer             | Description                                          |
|-------------------+------------------------------------------------------|
| =codex=           | OpenAI Codex CLI integration with project context.   |
| =direnv=          | direnv support via `envrc'.                          |
| =err-commonlisp=  | Common Lisp LSP (cl-lsp) and Flycheck setup.         |
| =err-core=        | Core err settings: LSP, TS/JS, Markdown, Org, UI.    |
| =err-ts=          | TypeScript/JavaScript prettier/node-modules tweaks.  |
| =llm=             | Large language model client configuration (gptel, ellama, MCP). |
| =obsidian=        | Obsidian vault integration for markdown buffers.     |
| =promethean-lisp= | Personal Promethean Lisp major mode.                 |
| =promethean-vterm= | vterm integration.                                  |
| =semantic-search= | Semantic search with sem.el + consult fallbacks.     |
| =unique-files=    | Timestamped unique files per mode + list continuation. |

* Adding a new layer

Create a subdirectory with at least a =packages.el= file.  See the
[[https://develop.spacemacs.org/doc/LAYERS.html][Spacemacs layer documentation]] for the full structure.
