#+TITLE: err-helm-popup Layer
#+AUTHOR: err
#+DATE: 2026-06-12

Private Spacemacs layer that runs selected Helm commands in a small temporary
floating frame.

* Commands

| Binding     | Command                      | Description                              |
|-------------+------------------------------+------------------------------------------|
| ~SPC o h f~ | `err/helm-popup-find-files`  | Floating Helm find files                 |
| ~SPC o h b~ | `err/helm-popup-list-buffers`| Floating Helm buffers list               |
| ~SPC o h r~ | `err/helm-popup-recentf`     | Floating Helm recent files (projectile)  |
| ~SPC o h p~ | `err/helm-popup-switch-project` | Floating Helm project switch          |

* Customization

Frame parameters are stored in `err-helm-popup-frame-params`.  Bindings are
under the `SPC o h` prefix.
